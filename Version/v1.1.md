# Version 1.1

Last updated: 2026.05

## Updates

- Added GitHub OAuth based admin login for the homepage editor.
- Added a Cloudflare Worker backend for admin authentication, GitHub content publishing, profile photo uploads, and visitor analytics.
- Added anonymous visitor tracking for the homepage.
- Added admin visitor analytics with total visits, unique visitors, selected-day visits, recent visit details, location, referrer, browser, and device.
- Replaced the visitor archive dropdown with a date picker and a Today button in the admin portal.
- Added support for multiple email addresses in homepage content and admin editing.
- Added profile photo upload support through the admin editor.
- Updated homepage content, links, projects, and profile photo through `content.json`.

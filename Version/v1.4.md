# Version 1.4

Last updated: 2026.05

## Updates

- Analytics: changed visit counting so repeated refreshes from the same visitor on the same path within 30 minutes do not create new visit events.
- Analytics: added a browser-side tracking window to avoid sending duplicate refresh events.
- Analytics: added a Worker-side dedupe guard so duplicate events are ignored even if the browser sends them.
- Content: bumped displayed version to `1.4`.

## Changed Files In Latest Update

- `script.js`
- `worker/src/index.js`
- `index.html`
- `content.json`
- `Version/v1.4.md` (new)

## Notes

- The visit window is currently 30 minutes.
- A new visit is still counted when the same visitor returns after the window, or when they visit a different path.
